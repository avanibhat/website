# Avani Bhat - Personal Website

A modern, responsive personal website showcasing my work as a Data Scientist specializing in applied analytics and healthcare data.

## Features

- Clean, modern design inspired by contemporary portfolio sites
- Multi-page structure with separate pages for each section
- Fully responsive layout (mobile, tablet, desktop)
- Modern typography using Space Grotesk font
- Interactive sections across multiple pages:
  - **Home** (`index.html`) - Introduction and hero section
  - **About** (`about.html`) - Professional summary
  - **Experience** (`experience.html`) - Work history and positions
  - **Projects** (`projects.html`) - Project showcases with GitHub links
  - **Education** (`education.html`) - Educational background and technical skills
  - **Contact** (`contact.html`) - Contact information and social links

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Space Grotesk)

## Getting Started

Simply open `index.html` in a web browser to view the website locally.

For development, you can use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then navigate to `http://localhost:8000` in your browser.

## Deployment

This website can be easily deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Customization

To customize the website:
1. Update content in `index.html`
2. Modify colors and styling in `styles.css` (CSS variables at the top)
3. Add interactive features in `script.js`

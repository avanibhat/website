Neuroquery — Alzheimer's Disease Research Assistant

2025

Built a Retrieval-Augmented Generation (RAG) system for querying Alzheimer's disease research literature. Uses BioBERT for biomedical-domain embeddings, ChromaDB for semantic vector search, and Groq (Llama 3.3 70B) for grounded, source-cited answer generation. Ingests PubMed abstracts via the Entrez API and local PDFs, served through a FastAPI backend with a vanilla JS chat interface.
